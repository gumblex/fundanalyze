# fundanalyze
Analyze and make a portfolio of funds.

## Dependencies

* bs4
* pulp
* numpy
* pandas
* PyPortfolioOpt

## Usage

`python3 fundvalue.py funds.db [fundid ...]`

Fetch data from http://fund.eastmoney.com/

`python3 portfolio.py funds.db [fundid ...]`

Calculate better portfolio allocations
